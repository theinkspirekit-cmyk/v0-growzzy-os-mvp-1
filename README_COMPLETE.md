## 3-Step Launch Process

### Step 1: Prepare (10 min)
```bash
# Copy env template
cp .env.local.example .env.local

# Add your API keys:
# - Supabase URL & key
# - OpenAI API key
# - OAuth credentials (Meta, Google, LinkedIn)
```

### Step 2: Test Locally (15 min)
```bash
npm install
bash setup.sh
npm run dev
# Visit http://localhost:3000
```

### Step 3: Deploy (15 min)
```bash
git push origin main
# Go to vercel.com
# Add environment variables
# Deploy
```
